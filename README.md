# Breakout
### Project Introduction
Breakout is based on the classic arcade game, Atari Breakout.
### What the Project Does?
The player will control a paddle to bounce a ball towards an array of bricks to destroy them and gain score in the process. A leaderboard is also included, saving your score along with a username and password.
### How to Run the Program
BreakoutApp.java is the main file and should be run as such on any platform the supports JavaSwing.
### Project Goals
To use the skills learnt in class to edit and improve AI generated code into a personalized game. Furthermore, collision detection, buttons and mouse events, abstract classes and inheritance, and GUI design + files I/O should be and were included.
### Setup Intructions
If save editing is required, maintain the following format in the UserInfo.txt file for each line: username,password,score,time. Additionally, the file requires an empty line at the bottom.
